COM ESSA API VOCE TEM A POSSIBILIDADE DE CRIAR SEU DECK CONFORME A COR DO SEU COMANDANTE ESCOLHIDO PREVIAMENTE E TAMBEM IMPORTAR UM DECK QUE VOCE TENHA CRIADO

 - PRIMEIRAMENTE NA ROTA ( http://localhost:8080/users/create ) voce insere os seus dados: nome, email, senha e role - e cria o seu usuario

 - EM SEGUIDA NA ROTA ( http://localhost:8080/users/login ) VOCE INSERE SEU DADOS: NOME, EMAIL E SENHA - PARA LOGAR E UTILZIAR O SISTEMA, APÓS FAZER O LOGIN
        SERÁ GERADO O TOKEN DE VALIDAÇÃO PARA PODER UTILIZAR AS OUTRAS FUNCIONALIDADES - LEMBRANDO QUE SEM O TOKEN NÃO É POSSIVEL UTILIZAR AS OUTRAS ROTAS

 -INFORME O TOKEN NA ABA AUTH - BEARER TOKEN

 - APÓS LOGIN, NA ROTA ( http://localhost:8080/deck/commanders ) VOCE PODE VER UMA LISTA DE TODAS AS CRIATURAS LENDARIAS QUE PODEM SER COMANDANTES
        ESCOLHA UM PARA SER O COMANDANTE DO SEU DECK - LEMBRANDO DE USAR O TOKEN DE VALIDAÇÃO

 - EM SEGUIDA NA ROTA ( http://localhost:8080/deck/commander/{NOME_COMANDANTE} ) VOCE IRA FORNECER NO FINAL DA URL O NOME COMPLETO DO COMANDANTE ESCOLHIDO
        PARA SER SALVO NO BANCO E NO ARQUIVO JSON

 - APÓS SALVAR SEU COMANDANTE VOCE IRA BUSCAR AS CARTAS QUE PODEM SER UTILIZADAS NO SEU DECK DE ACORDO COM A COR DO COMANDANTE
        NA ROTA ( http://localhost:8080/deck/cards ) VOCE GERA AUTOMATICAMENTE SEU DECK CONTENDO 99 CARTAS QUE PODEM SER USADAS NO SEU DECK
        *ESSA BUSCA TAMBEM TRAZ O COMANDANTE POIS ELE FAZ PARTE DO SEU DECK* TOTALIZANDO AS 100 CARTAS PARA COMPLETAR SEU DECK DE MAGIC


 TODAS AS CARTAS SÃO SALVAS NO BANCO DE DADOS (POSTGRES) E TAMBEM NO ARQUIVO JSON CRIADO NO PROJETO

 - ALEM DE CRIAR O DECK CONFORME O COMANDANTE ESCOLHIDO, VOCE PODE IMPORTAR UM DECK QUE TENHA CRIADO ANTERIORMENTE

 - NA ROTA ( http://localhost:8080/deck/import ) VOCE PODE INSERIR AS CARTAS DO SEU DECK PARA ADICIONAR NO BANCO DE DADOS



 - OS COMANDANTES SÃO ADICIONADOS NO BANCO (commander)
 - AS CARTAS DO DECK GERADO PELO COMANDANTE É ADICIONADO NO BANCO (cards)
 - AS CARTAS DO DECK IMPORTADO SÃO ADICIONADAS NO BANCO (imported_cards)


- O PROGRAMA UTILIZA RABBITMQ PARA TROCA DE MENSAGENS NA CRIAÇÃO DO DECK OU IMPORTAÇÃO DE UM DECK
    POR ISSO DEVE SE UTILIZAR O COMANDO ABAIXO PARA LINKAR COM O DOCKER, O DOCKER DESKTOP E O CONTAINER CRIADO
    DEVE ESTAR RODANDO NA MAQUINA PARA O PROGRAMA FUNCIONAR

 - docker run -d --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:management
